WeCare App

My First iOS App

Due to contract restrictions I am unable to share the final build of the app. However, the initial build and its code can be found on my gihub if you would like to peruse.
The app was designed to be used to track and monitor patient health and nurse activity for the company. It was built for iOS using swift/objectiveC.

The main goal of the UI was to be clean, minimal, and ofcourse easy to use. There was limited use of any custom animations to keep the app as efficient as possible.
A FireBase backend was setup to hold patient and nurse information in a NoSQL database. This was my first introduction with a NoSQL backend and therefor it was a challenge to construct it. However, google's firebase team provides great documentation to help any beginer set up a database that any veteran architect would be proud of.

The app/ firebase backend, kept track of nurse location and their activity, as well as patient vitals initially by manual reporting, later updated to automatic reporting from fitness tracker API's.
