WeCare Website

Have a look at the website here

After dabbling with web development for a few weeks, I was confident enough to pitch a complete redesign for my employers website.
The current website was incomplete and barely functional, but it conveyed the desired message.

My goal was to not only rebuild the website, but get some of the core features working.
Features such as contact form submission, handling, and page routing (which was missing in the original build).
Along with this I also wasnted to improve the UI to a much cleaner and modern look that got the point across as succinctly as possible.

At this point my stack for web development consisted of HTML, CSS and JavaScript. My limited knowledge coupled with lofty ambitions for this site, forced me to come up with unorthodox methods that yielded simple and effective results.
Specifically, instead of dealing with php server difficulties when dealing with form submissions, I found a simpler method of using a google script linked to the form to handle the sending of information from the client to my employer and simultaneously updating a google Sheet.

Long story short, this project got me really interested in the capabilities yet to be discovered of the world of web development. Feel free to inspect away at the page and any feedback would be appreciated! Cheers
